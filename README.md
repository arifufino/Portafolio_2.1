# Portafolio 2.1

Portafolio profesional desarrollado con React y Vite para mostrar información personal, proyectos, habilidades y experiencia como desarrollador de software.

## Características

- Diseño completamente responsivo.
- Animaciones y transiciones suaves.
- Sección Sobre Mí.
- Proyectos destacados.
- Tecnologías y habilidades.
- Información de contacto.
- Integración con GitHub.
- Interfaz moderna y personalizable.

## Tecnologías Utilizadas

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Git
- GitHub

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/arifufino/Portafolio_2.1.git
```

Entrar al proyecto:

```bash
cd Portafolio_2.1
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

Construir para producción:

```bash
npm run build
```

## Variables de entorno

Crear un archivo:

```env
.env
```

Tomando como referencia:

```env
.env.example
```
